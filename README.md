# ProyectoSGC
Este proyecto es un sistema de gestión para Cooperativas

## Instrucciones de inicio
Para ingresar como Administrador, el usuario que se crea por defecto es:
- Usuario: `administrador`
- Contraseña: `123456`

El usuario administrador puede crear también un usuario tipo Socio para probar las pantallas y funcionalidades del socio.
